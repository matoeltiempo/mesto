# Project "Express"

### Установка и запуск проекта
1. Скачайте сборку архивом или используя команду:
```git clone git@github.com:matoeltiempo/mesto.git```
2. Запустить установку через терминал:
```npm i```
3. Выбрать неообходимый вариант сборки:
```
# production
npm run start
# develop
npm run dev
```
